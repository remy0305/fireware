 ===========================Steps for porting BMD31M090 Module to HT32 Starter Kit.===========================
 Step 1. Modify example_xxxxx\_ProjectConfig.ini.
         - Add DEVICE_NAME
         Example [take HT32F52352 as an example]:
           REM DEVICE_NAME
           REM ============================================================================
           52352

 Step 2. Modify example_xxxxx\ht32_board_config.h.
         - Add module communication interface pin setting
         Example [take HT32F52352 as an example]:
            #if defined(USE_HT32F52352_SK)
            /*----------------------------------------------------------------------------------------------------------*/
            /* I2C Master Setting                                                                                       */
            /*----------------------------------------------------------------------------------------------------------*/
              #undef  BMD31M090_WIRE                               // Starter Kit Demo Using WIRE, so NEED TO #undef first.
              #define BMD31M090_WIRE                          (0)  // Default Using WIRE for Starter Kit.

              #define HTCFG_I2CM_CH0                          I2C1
              #define HTCFG_M0_SCL_GPIO_PORT                  A
              #define HTCFG_M0_SCL_GPIO_PIN                   0
              #define HTCFG_M0_SDA_GPIO_PORT                  A
              #define HTCFG_M0_SDA_GPIO_PIN                   1
            #endif

 ===========================Steps for porting BMD31M090 Module to Another Vendor's MCU.=======================
 Step 1. Unzip File:
         - Start by unzipping "BMD31M090_HT32_Vm.n.r.zip".

 Step 2. Port Source Code:
         - Migrate the src directory to your target project.
         Example: D:\<Another Vendor's MCU Target Project>\src

 Step 3. Include in Compilation:
         - Open the Keil project and make modifications as below.
         (Ensure "BMD31M090.c" and header files are included in your project's build list.)
         1. Add the module "Include Paths" to the project.
            Example: D:\<Another Vendor's MCU Target Project>\src
                     D:\<Another Vendor's MCU Target Project>\src\utility
         2. Add the module "C language files" to the project.
            Example: BMD31M090.c
                     fontTable.c

 Step 4. Modify Includes and APIs:
         - Use global search to identify and modify include files and APIs for porting.
         /*
           !!! PORTING !!!
           Description...
         */
         Modify the following:
         1. Add driver header file
             BMD31M090.c
             BMD31M090.h
         2. Declare I2C structure variable in BMD31M090.c
         3. Pay attention to the Slave address of BMD31M090.
             7-bit: 0x3C
             8-bit: 0x78 or 0x79
         4. Update I2C initial function
         5. Update I2C master write function
             modification :
                 BMD31M090_selWire()
                 BMD31M090_Init()
                 setCommand(uint8_t command)
                 setData(uint8_t command)
