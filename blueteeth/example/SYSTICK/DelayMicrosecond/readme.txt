@par Example Description:

This example show how to delay microsecond by SYSTICK.
 
@par Directory Contents:

- SYSTICK/DelayMicrosecond/main.c               Main program
- SYSTICK/DelayMicrosecond/ht32fxxxx_it.c       Interrupt handlers
- SYSTICK/DelayMicrosecond/ht32_board_config.h  Board configuration file

@par Hardware and Software Environment:

- This example can be run on HT32 Series development board.
- Refer "ht32_board_config.h" for pin assignment.

@note

The PA0(default) will be toggled from 1 uS ~ 12 uS.
