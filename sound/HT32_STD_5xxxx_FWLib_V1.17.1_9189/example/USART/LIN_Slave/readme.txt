@par Example Description:

This example shows how to configure USART/UART as LIN Slave mode to send and receive data.

@par Directory Contents:

- USART/LIN_Slave/main.c                          Main program
- USART/LIN_Slave/ht32fxxxxx_nn_it.c              Interrupt handlers
- USART/LIN_Slave/ht32_board_config.h             Board configuration file

@par Hardware and Software Environment:

- This example can be run on the HT32 series development kit.
- Refer "ht32_board_config.h" for pin assignment.
