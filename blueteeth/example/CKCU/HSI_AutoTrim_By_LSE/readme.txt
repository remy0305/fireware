@par Example Description:

The example shows how to use the HSI Auto Trim function by LSE.

@par Directory Contents:

- CKCU/HSI_AutoTrim_By_LSE/main.c                           Main program
- CKCU/HSI_AutoTrim_By_LSE/ht32fxxxxx_nn_it.c               Interrupt handlers

@par Hardware and Software Environment:

- This example can be run on the HT32 series development kit.
- You need a terminal software such as TeraTerm (https://ttssh2.osdn.jp/).
- Connect a null-modem female/female RS232 cable between the UxART and PC serial port.
  Terminal configuration:
  - Word Length = 8 Bits
  - One Stop Bit
  - No parity
  - BaudRate = 115200 baud
  - Flow control: None
