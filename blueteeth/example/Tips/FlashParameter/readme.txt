@par Example Description:

This example describes how to operation liner script to specify the memory layout,
storing code and parameters in different memory areas.

@par Directory Contents:

- Tips/FlashParameter/main.c                             Main program
- Tips/FlashParameter/ht32fxxxx_it.c                     Interrupt handlers
- Tips/FlashParameter/flash_parameters.c                 The flash parameters source file
- Tips/FlashParameter/flash_parameters.h                 The flash parameters header file
- Tips/FlashParameter/linker.lin                         The flash parameters scatter-loading file

@par Hardware and Software Environment:

- This example can be run on HT32 Series development board.
